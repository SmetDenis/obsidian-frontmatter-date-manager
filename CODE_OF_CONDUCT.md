# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its Code of Conduct. The full, authoritative text is maintained at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## In short

- Be respectful and considerate; assume good intent.
- Welcome newcomers and differing viewpoints, and keep criticism constructive.
- Harassment, personal attacks, and other unwelcome conduct are not tolerated.

## Scope

This Code applies in all project spaces (issues, pull requests, discussions) and when an individual is representing the project in public.

## Reporting

Report unacceptable behavior **privately** to the maintainer using the contact methods listed on the [@SmetDenis](https://github.com/SmetDenis) GitHub profile. Please do not use public issues for reports. All reports are handled confidentially.

Enforcement follows the Contributor Covenant guidelines: a private warning for minor issues, escalating to a temporary or permanent ban for serious or repeated violations.
